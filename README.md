# 🎭 AI Face Detection System

An AI-powered Face Detection System built using **Python, OpenCV DNN (SSD), and Streamlit**. The application detects multiple faces in images and real-time webcam streams, draws bounding boxes, displays confidence scores, and provides an interactive web interface.

---

## 🚀 Features

- 📸 Detect faces in uploaded images
- 🎥 Real-time webcam face detection
- 👥 Detect multiple faces simultaneously
- 📊 Display confidence score for each detected face
- 🔢 Show total number of detected faces
- 🎚️ Adjustable confidence threshold
- 🌐 Interactive Streamlit web application

---

## 🛠️ Tech Stack

- Python
- OpenCV
- OpenCV DNN (SSD Face Detector)
- NumPy
- Streamlit

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/rayalJiya/AI-Face-Detection-System.git
```

Move into the project directory:

```bash
cd AI-Face-Detection-System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

### Detect faces in an image

```bash
python detect.py
```

### Real-time webcam detection

```bash
python webcam.py
```

### Streamlit Web App

```bash
streamlit run app.py
```
